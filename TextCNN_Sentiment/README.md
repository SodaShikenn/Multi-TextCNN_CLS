### Pytorch Bert_TextCNN 新闻文本分类项目

文本分类是自然语言处理最基础，也是最重要的任务之一，非常适合作为NLP入门的第一个项目。本项目数据集为清华大学的 THUCNews 新闻文本分类数据集（子集），公训练集18w，验证集1w，测试集1w，10个类别：金融、房产、股票、教育、科学、社会、政治、体育、游戏、娱乐。
